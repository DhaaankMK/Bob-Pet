# assets/sprites/
# Pasta para sprites personalizados do Bob (opcional)
# 
# O Bob usa desenho procedural por padrão (sem sprites).
# Para usar sprites PNG personalizados, adicione arquivos aqui no formato:
#
#   bob_idle_000.png, bob_idle_001.png, ...   → Idle (quantos frames quiser)
#   bob_walk_000.png, bob_walk_001.png, ...   → Caminhada
#   bob_jump_000.png                          → Pulo
#   bob_fall_000.png                          → Queda
#   bob_sleep_000.png, bob_sleep_001.png, ... → Dormir
#   bob_dance_000.png, bob_dance_001.png, ... → Dança
#
# Tamanho recomendado: 100x130 pixels, fundo transparente (PNG)
# O sistema carregará automaticamente se encontrar esses arquivos.
